About Me

Full Stack Developer!

Developed a secure and scalable booking system using .NET Core, featuring user authentication with Identity Framework. Implemented registration, login, forgot password, and social login via Google and Facebook. Added SMTP-based email verification and role-based access control to ensure secure access to booking features.

Users can browse hotels and restaurants, make reservations, and manage their bookings. The admin panel allows efficient management of users, bookings, and listings. The application is deployed on Azure Web Services for enhanced scalability, security, and high availability.

Technologies: .NET Core, Identity Framework, SQL Server, Entity Framework, Razor Pages/MVC, SMTP, Google & Facebook Authentication, HTML, Bootstrap, Azure Web Services.
